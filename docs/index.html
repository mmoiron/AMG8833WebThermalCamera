AMG8833 Web Thermal Camera

Web-based firmware installer for Wemos D1 Mini

1 Flash Firmware

Connect your Wemos D1 Mini via USB and click the button below. This will install both the firmware and the web UI filesystem.

Browser Not Supported

Requirements

  • Chrome or Edge browser (desktop)
  • Wemos D1 Mini connected via USB
  • CH340 USB driver installed

2 Wire the Sensor

Connect the AMG8833 IR sensor to the Wemos D1 Mini:

Wemos D1 MiniAMG8833
3V3VIN
GNDGND
D2 (GPIO4)SDA
D1 (GPIO5)SCL

3 Connect & View

After flashing and wiring, the thermal camera is ready to use:

Quick Start

  1. Power the Wemos D1 Mini
  2. On your phone, connect to WiFi THERMAL_ESP (password: thermal1234)
  3. Open browser and go to http://192.168.4.1
  4. You should see the live thermal heatmap
